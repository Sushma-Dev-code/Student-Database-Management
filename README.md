# 🎓 Student Database Management System

A full-stack web application for managing student records.

## 🚀 Features

- User Login (JWT Authentication)
- Dashboard
- Add Student
- View Students
- Edit Student
- Delete Student
- Restore Deleted Students

## 🛠️ Technologies Used

### Frontend
- Angular
- TypeScript
- HTML
- CSS

### Backend
- Node.js
- Express.js
- JWT Authentication

### Database
- PostgreSQL

## 📂 Project Structure

```
Student-Database-Management
│
├── Backend
├── frontend
└── README.md
```

## ▶️ Run the Project

### Backend

```bash
cd Backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
ng serve
```
